## Instructions

Given two integers `a` and `b`, return the sum of the two integers without using the `+` and `-` operators.

**Example 1**:

```Java
Input: a = 1, b = 1

Output: 2
```

**Example 2**:

```Java
Input: a = 4, b = 7

Output: 11
```

**Constraints**:

- `-1000 <= a, b <= 1000`
